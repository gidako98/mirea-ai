# Data

The project uses synthetic support ticket data generated by `src/data.py`.
The sample file `sample_tickets.csv` is small enough for quick checks and does not contain personal or confidential information.

Columns:

- `ticket_id` — synthetic ticket identifier;
- `text` — short support-ticket text;
- `channel` — source channel: email, chat, phone, web;
- `customer_tier` — synthetic customer tier: free, standard, premium, enterprise;
- `product_area` — affected product area;
- `sentiment_score` — numeric sentiment-like score from -1 to 1;
- `account_age_days` — synthetic account age;
- `priority` — target label: low, medium, high.

Regenerate data:

```bash
python -m src.data --output data/sample_tickets.csv --rows 600 --seed 42
```
